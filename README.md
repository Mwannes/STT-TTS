وقمت بدمج الاوامر اللازمه مع الكود ليتم حفظ النص كملف نصي 
الخطوة التالية هي تحويل النص الى صوت ويقوم بقراءة النص الناتج من الخطوة الاولى ويحفظ هذ الملف كملف صوتي بصيغة MP3
